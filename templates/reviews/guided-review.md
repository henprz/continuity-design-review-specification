# Guided Review

Guided Reviews are intended for situations where additional evidence can be collected before evaluation.

Characteristics

- Moderate evidence.
- Clarifying questions are encouraged.
- Medium confidence.

Before beginning the review, the reviewer should identify the most valuable missing evidence and invite the user to provide it.

The generated report MUST follow the official report template.

Differences from other review modes:

- More context hypotheses can be validated.
- Assumptions should decrease as evidence grows.
- Confidence should increase only when supported by evidence.
